# نظام إدارة الفلج (Falaj Management System)

نظام إلكتروني متكامل لإدارة وتنظيم جداول سقي الأفلاج التقليدية، مبني باستخدام HTML و Tailwind CSS و React (بدون الحاجة لخوادم معقدة، يعمل مباشرة في المتصفح).

## الميزات الأساسية 🌟
- 🕒 **شاشة مراقبة حية:** عرض من يسقي الآن مع عداد تنازلي دقيق، ومن هو التالي في الجدول.
- 🔔 **نظام الإشعارات:** تنبيهات عبر المتصفح وصوت رنين قبل 15 دقيقة من دورك وعند بدايته.
- 📱 **متجاوب تماماً:** تصميم يعمل بشكل ممتاز على الهواتف المحمولة والشاشات الكبيرة.
- 👤 **جدولي:** محرك بحث سريع يتيح لأي مساهم البحث باسمه أو رقمه لمعرفة أوقات سقيته والاشتراك في تنبيهاتها.
- 🛠️ **أدوات الفلج:** 
  - أداة طلب مبادلة الردة مع شخص آخر وتوليد رسالة واتساب جاهزة لوكيل الفلج.
  - عرض أوقات الصلاة والشروق والغروب لارتباطها الوثيق بتوقيت الفلج.
- 🛡️ **لوحة الإدارة:** لوحة تحكم محمية بكلمة مرور لإضافة، تعديل، وحذف الجداول والسجلات.

## طريقة التشغيل 🚀

النظام لا يتطلب أي تنصيب، فقط افتح ملف `index.html` في أي متصفح حديث، وسيعمل بشكل كامل.

> **ملاحظة:** لتعمل ميزة الإشعارات (Web Notifications)، يُفضل تشغيل الملف عبر خادم محلي (Local Server) وليس فقط بفتحه كمستند `file://` نظراً لسياسات الأمان في المتصفحات الحديثة.
> يمكنك تشغيله محلياً عبر بايثون:
> `python -m http.server 8080`
> ثم الدخول إلى `http://localhost:8080`

## بيانات الدخول الافتراضية للوحة الإدارة 🔒
- **كلمة المرور:** `Asdf@#$1234`

## تقنيات الاستخدام 💻
- **HTML5 / CSS3**
- **React 18** (مُدمج عبر CDN)
- **Babel** (لترجمة JSX في المتصفح)
- **LocalStorage** (لحفظ البيانات محلياً بدون الحاجة لقاعدة بيانات)

## المساهمة والتطوير 🤝
يمكن تطوير هذا الملف ليصبح تطبيق ويب متكامل عبر أطر عمل مثل Next.js أو ربطه بقاعدة بيانات حقيقية (مثل Firebase أو Supabase) لمشاركة الجداول بين جميع المشتركين لحظياً.
